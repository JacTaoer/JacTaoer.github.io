JS-asyncawait

发表于2023-06-04|更新于2023-06-05|前端基础JS进阶

1、async / await

  • 异步回调 callback hell
  • Promise then catch 链式调用，但也是基于回调函数
  • async / await 是同步语法， 彻底消灭回调函数

2、async / await 和Promise 的关系

  • async / await 是消灭异步回调的终极武器

  • 但和Promise 并不互斥

  • 反而，两者相辅相成

    两者结合

    • 执行async 函数 ，返回的是Promise对象
    • await相当于Promise 的 then
    • try… catch 可捕获异常，代替了Promise 的catch

3、异步的本质

  • async/ await 是消灭异步回调的终极武器

  • JS还是单线程，还得是有异步，还得是基于event loop

  • async / await 只是一个语法糖
